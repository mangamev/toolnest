# ToolNest - Free Online Tools

A collection of free, browser-based utility tools. No signup required, no data sent to servers.

## Tools Included

1. **Word Counter** – Count words, characters, sentences, paragraphs
2. **Case Converter** – UPPER, lower, Title, Sentence, camelCase, snake_case
3. **Password Generator** – Cryptographically secure, customizable
4. **Base64 Encoder/Decoder** – Encode and decode Base64 strings
5. **JSON Formatter** – Prettify and minify JSON with validation
6. **Color Picker** – HEX, RGB, HSL converter
7. **QR Code Generator** – Generate and download QR codes
8. **Age Calculator** – Calculate exact age with days breakdown
9. **Random Number Generator** – Generate single or multiple random numbers
10. **URL Encoder/Decoder** – Encode and decode URLs

## Deployment

This site is deployed via GitHub Pages.

1. Push all files to a GitHub repository
2. Go to Settings → Pages → Branch: main → Save
3. Site will be live at `https://yourusername.github.io/repo-name`

## Files

- `index.html` – Main page with all tools
- `privacy.html` – Privacy Policy (required for AdSense)
- `terms.html` – Terms of Use
- `about.html` – About page
