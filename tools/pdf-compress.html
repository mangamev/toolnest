PDF Compressor

Reduce PDF file size by rebuilding the document's internal structure and stripping unused metadata. Runs entirely in your browser — nothing is uploaded.

-Original size
-Pages

This rebuilds the PDF with a single, compressed cross-reference table and removes title/author/metadata — this can meaningfully shrink PDFs that have been edited many times in desktop software. For PDFs whose size is dominated by large images, the reduction will be small, since image data itself isn't recompressed.
