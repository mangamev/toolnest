PDF 페이지 관리자

여러 PDF를 하나로 합치거나, 페이지 순서를 바꾸거나 삭제해서 분할하고, 개별 페이지를 회전한 뒤 내보내세요. 모든 처리는 브라우저에서 실행됩니다.

PDF 한 개를 업로드하면 페이지 순서 변경, 회전, 삭제(간단한 분할 방법)가 가능합니다. 여러 PDF를 업로드하면 업로드한 순서대로 합쳐집니다.

✕는 해당 페이지를 결과에서 제외합니다(간단한 삭제·분할 방법). ↑/↓로 페이지 순서를 바꿉니다. ↺/↻는 페이지를 90도 회전합니다. 일부 페이지만 추출하려면 나머지를 모두 제외하고 내보내세요.
